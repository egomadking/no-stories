about this site

Recipe stories can be tedious.

Sometimes the only thing needed is a recipe. This site is an uncluttered reference for those who need a to see a recipe while cooking. These recipes are as well-crafted as any tale.

About me

I grew up in the 80's and 90's but self-identify as an elder millennial. Married, 1 kid, cats. Military retiree. I'm a fan of flat caps and cast iron. I'm highly inspired by the cuisines we experienced while we traveled abroad. Cucina povera of southern Italy, germanic dishes, and hungarian foods all have a special place in my heart.

Not a profit-driven site

Hosting is cheap these days so this site is a personal playtoy. No one is paying for this site other than myself. Links or brands that appear on this site only because of a personal preference. If any of this were ever to change, it will be noted here.

Geekery

I am using Eleventy to build this site and hosting on Netlify. No frameworks were used because none were needed; there are several CSS resets in place but all other code on this site is my own devising. For those who are curious, you can see the site's Github repo here: egomadking/no-stories
